# 🐍 Python OOP (Class va Object) Amaliyot Repozitoriysi

Ushbu repozitoriy Python dasturlash tilida **Object-Oriented Programming (OOP)** bo‘yicha barcha amaliy mashqlar va mini loyihalarni o‘z ichiga oladi.  
Maqsad — class, object va real hayotga yaqin tizimlar yaratish orqali dasturlash ko‘nikmalarini rivojlantirish.

---

## 📌 Repozitoriy haqida

Bu loyiha Python OOP konseptlarini bosqichma-bosqich o‘rganish uchun yaratilgan.  
Har bir class real hayotdagi tizimlarni simulyatsiya qiladi va mantiqiy fikrlashni rivojlantiradi.

---

## 📚 O‘rganilgan mavzular

- Class va Object
- `__init__` konstruktor
- Instance attributes
- Instance methods
- Getter va setter tushunchasi
- if/else bilan logika yozish
- State (holat) boshqaruvi
- Real hayotga yaqin modellar
